Configure SMTP email sending for this Flask app on Replit.

Project context:
- Backend file: backend/server.py
- Email mode uses SCHIROPDF_EMAIL_MODE=smtp
- SMTP env vars required by app:
  - SMTP_HOST
  - SMTP_PORT
  - SMTP_USERNAME
  - SMTP_PASSWORD
  - SMTP_FROM_EMAIL

Tasks:
1) Open Replit Secrets and set these values:
   - SCHIROPDF_EMAIL_MODE=smtp
   - SMTP_HOST=<provider host>
   - SMTP_PORT=587
   - SMTP_USERNAME=<smtp username>
   - SMTP_PASSWORD=<smtp password>
   - SMTP_FROM_EMAIL=<verified sender>
2) Ensure backend starts with these secrets available.
3) Test endpoints:
   - POST /api/auth/register (non-admin) should return email_delivery="sent" and no dev_verification_code.
   - POST /api/auth/request-password-reset should return email_delivery="sent" and no dev_reset_code.
4) If provider blocks SMTP, switch temporarily to:
   - SCHIROPDF_EMAIL_MODE=dev
   then confirm app still works with dev codes.
5) Output a short deployment report including:
   - Which SMTP provider worked
   - Any provider-specific settings needed (TLS, app passwords, sender verification)
   - Final Secrets checklist
